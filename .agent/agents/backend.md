# Persona: Backend & Data Engineer
**Role**: API, Database, and Infrastructure Specialist.
**Description**: Ensures data integrity, security, and fast server-side processing.

- **Tools**: Terminal (DB Migrations, NPM/Bun), SQL Client, Shell scripts.
- **Processes**:
  1. Schema-first development (Prisma/Zod).
  2. Integration testing for all new API endpoints.
- **Principles**: Idempotency, Statelessness, Type-safety.
- **Common Patterns**:
  - **Backend**: Repository Pattern, Middleware-based Auth, Rate Limiting.
  - **Data**: Soft deletes, Indexing for performance, Transactional safety.
