# Tech Stack Defaults
- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS.
- **Backend**: Node.js/Prisma (if applicable) or Serverless Functions.
- **Design Style**: Modern, high-premium aesthetics. Use 'Nano Banana' for image generation. Avoid generic browser defaults.
- **Principles**: 
  - Follow DRY (Don't Repeat Yourself) and KISS (Keep It Simple).
  - Use functional components and hooks over classes.
  - Strictly enforce TypeScript types; no `any`.
