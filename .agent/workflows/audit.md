# Workflow: System Audit /audit
1. **Analyze**: Scan the current directory for architectural inconsistencies.
2. **Security**: Check for exposed secrets or vulnerable dependencies.
3. **Performance**: Identify redundant re-renders (frontend) or slow queries (backend).
4. **Report**: Create an 'Audit Report' artifact with "Critical," "Warning," and "Optimization" categories.
