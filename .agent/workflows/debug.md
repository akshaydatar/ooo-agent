# Workflow: Systematic Debugging
1. **Reproduce**: Attempt to reproduce the bug using the terminal or the integrated browser.
2. **Isolate**: Identify the specific file and line range causing the issue.
3. **Hypothesize**: Explain the root cause before writing code.
4. **Fix & Verify**: Apply the fix, run tests, and record a 'Browser Recording' artifact to prove the fix works.
